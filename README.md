# Base Ideas
1. Debt spiral occurs if the economic growth outpaces the interest it pays on it debts

Δd=(r−g)d−pb 
r = Effective real interest rate on government debt
g = real gdp growth rate
d = Total debt/gdp ratio
pb = Primary Balance/ GDP ratio (government revenue minus non-interest spending. )

From the above, if real interest rate is higher than economic growth rate, debt ratio will grow exponentially unless the government runs a massive surplus. if r-g is positive and monotonically increasing, no amount of intervention can save it.
